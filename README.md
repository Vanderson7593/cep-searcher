## Hight Level Used Technologies

## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
```
## Running the app

Run the development server:

```bash
npm run dev
# or
yarn dev
```
Then access your localhost in 3000 port http://localhost:3000
